Welcome to AiiDA-ShengBTE's documentation!
==========

AiiDA-ShengBTE is a plug-in for the workflow management and data provenance tracking framework `AiiDA`_. It provides the classes `AiiDA`_ needs to run simulations using `ShengBTE`_ .

`ShengBTE`_ is a solver for the Boltzmann transport equation for phonons. For detailed documentation on using `ShengBTE`_ take a look in their `ShengBTE wiki`_.

==========

.. _ShengBTE: http://www.shengbte.org/
